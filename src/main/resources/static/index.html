02

Technology Stack

☕
Java 17 LTS
Records, sealed classes, text blocks. Target runtime for production deployments. Required for Spring Boot 3.x.
🍃
Spring Boot 3.2.5
Jakarta EE 10 namespace. Auto-configuration, embedded server, production-ready defaults. WAR deployment support.
🐱
Apache Tomcat 10
Jakarta Servlet 6.0 container. Hosts the deployed WAR. External instance replaces Spring's embedded Tomcat.
⚙️
Apache Maven 3.x
Build lifecycle management: validate → compile → test → package → install → deploy. Produces todo-app.war.
🗄
Spring Data JPA + H2
Hibernate ORM with H2 in-memory database. Zero external DB required. Swap to PostgreSQL/MySQL for production.
❤
Spring Actuator
Health checks, metrics, and environment info at /actuator/*. Essential for monitoring in production environments.
✅
Bean Validation
Jakarta Validation API with @NotBlank, @Size annotations. Request body validation with automatic 400 responses.
🐳
Docker Ready
Dockerfile included for containerisation. Multi-stage build: Maven compile → Tomcat runtime image. CI/CD pipeline ready.

04

REST API Reference

GET /api/todos Retrieve all todo items
GET /api/todos/{id} Retrieve a single todo by ID
POST /api/todos Create a new todo (JSON body required)
PUT /api/todos/{id} Replace a todo entirely
DELETE /api/todos/{id} Delete a todo, returns 204 No Content
PATCH /api/todos/{id}/toggle Toggle completed status
GET /api/todos/stats Aggregate counts: total / completed / pending
GET /api/todos/filter?completed=true Filter by status or priority
GET /api/info Application metadata and runtime info
GET /actuator/health Spring Actuator health check (DB + disk)
GET /h2-console H2 database web console (dev only)

DevOps Engineer Notes

⚠️
Tomcat Version Matters
Spring Boot 3.x requires Tomcat 10+ (jakarta.* namespace). Tomcat 9 uses javax.* and will throw ClassNotFoundException at deployment.
🔒
Disable H2 in Production
Set spring.h2.console.enabled=false in production profile. Replace H2 with PostgreSQL or MySQL via spring.datasource.* properties.
📊
Actuator for Monitoring
Expose /actuator/health to load balancer health checks. Wire /actuator/metrics to Prometheus + Grafana for production observability.
🐳
Containerise This App
Use the included Dockerfile for a multi-stage build. Stage 1: Maven compile. Stage 2: Tomcat runtime. Final image ~180MB.
